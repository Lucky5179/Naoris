FAQ

How do I install the Testnet?
+

Testnet installation guides will be available when the Testnet is released planned for Q3. You will be informed when they are released. Please sign up here for early access.

How can I be eligible for an airdrop?
+

You will be eligible for an airdrop by running the Testnet for as long as possible and also referring other users. You will be informed when the Testnet is released. Please sign up here for early access

Are referrals possible?
+

Yes, referrals are possible, information on how to refer will be available when the Testnet is released. Please sign up here for early access.

How can I buy $NAORIS?
+

You will be able to buy $NAORIS tokens when the TGE takes place in Q4.

What is the utility of the $NAORIS token?
+

Please see $NAORIS Token in the Learn menu here or for a deeper understanding go to the Knowledge Base here.

How can you guarantee my privacy & anonymity?
+

Naoris Protocol's decentralized nature means that it operates by default as an integrated confidentiality infrastructure that assures privacy and anonymity. Your device generates an anonymous ID so the blockchain can recognize your node privately and rewards your participation.

Can I run a node?
+

Yes, the ability to run a node will be available from when mainnet goes live.

What are the system requirements for the Testnet security node and agent?
+

The first release of the Tesnet will be available for Linux, Windows and Mac OS.

Is the security node and agent safe to install?
+

Yes the security node and agent are safe to install. No personal or sensitive data is collected. Naoris Protocol collects only anonymized data crucial for enhancing collective cybersecurity intelligence. Contributions from nodes aid in refining the decentralized Swarm AI threat detection algorithm.

Will security nodes and agents conflict with any software or virus scanners?
+

Naoris Protocol operates on a separate decentralized cyber-layer, allowing already deployed cybersecurity tools to operate without conflicting with them.
